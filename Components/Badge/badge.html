Badge

Badges are used to highlight an item's status for quick recognition. It is generally used to show the count of notification or status of the user.

Status Badge

Status badge is used to show the status of the user. It has 3 variants online-badge | dnd-badge | idle-badge

Icon badge

Icon badge is used to show the count of the notification or add cart option. It can be created using the class badge-icons icon-badge Font size of badge text is in em. It will change as per the parent element's font-size.

Button with Badge

Button with badge can be created using the class button-badge-container icon-badge badge-of-btn. Font size of badge text is in em. It will change as per the parent element's font-size.

Toggle Button with badge

Toggle Button with badge can hide the batch on clicking on the button. It can be created using the class toggle-button-with-badge toggle-button-with-badge-id. Font size of badge text is in em. It will change as per the parent element's font-size.

Text with badge

Text with badge can be created using the class icon-with-badge text-with-badge-id. Font size of badge text is in em. It will change as per the parent element's font-size.
