Button

The Button component is used to trigger an action or event, such as submitting a form, opening a dialog, cancelling an action, or performing a delete operation.

Contained Button

Contained buttons are high-emphasis, distinguished by their use of elevation and fill.
There are 4 variants - Primary | Secondary | Success | Error
These can be created using class secondary success error. Primary button is the default button.

Outline Button

Outlined buttons are generally used in secondary purposes where more emphasis is not needed.
There are 4 variants - Primary | Secondary | Success | Error
These can be created using class outline-primary outline-secondary outline-success outline-error.

Link Button

Link buttons are used to add the link in the button. Use the class link-btn in the button

Icon Button

Icon button can be created using the class icon-btn and add the appropriate icon.

Social Button

Social button is used for the social media platforms. Use the class icon-btn

Floating Button

Floating button can be used to fix button in any space you want. Use the class floating-btn
