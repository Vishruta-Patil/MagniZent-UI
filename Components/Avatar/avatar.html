Avatar

The Avatar component is used to represent a user, and displays the profile picture, initials or placeholder icon. It can be used in login page, notification bar, and in various parts of the project etc

Avatar component comes in 6 sizes. You can change the size of avatar by adding the classes.
Classes available are xl-2 | xl | lg | md | sm | xs

Circular Avatar

Circular Avatars can be created using the class avatar inside the img tag.

Rounded Avatar

Rounded Avatars can be created using the class avatar-rounded inside the img tag.

Avatar with badge

Avatar with badge can be created using the class avatar badge-avatar. And badge can be created using the class badge badge-xl-2
Don't forget to add the size of badge according to the chosen avatar. And you can change the image by overwriting css of the class badge-avatar by adding the background image url.

Avatar with placeholder icon

Avatar with placeholder icon can be created using the class avatar-placeholder, inside the img tag.

Avatar with placeholder initial

Avatar with placeholder initial can be created using the class avatar-placeholder-initial, inside the img tag. And the class flex-container will be used to center the initial.
